Visión General
DevStock es un sistema de gestión para comercios minoristas que está evolucionando de una solución local a un modelo SaaS (Software as a Service) con inteligencia artificial integrada para la toma de decisiones estratégicas.

Stack Tecnológico
Frontend: Next.js (App Router).

Base de Datos: PostgreSQL.

UI Library: HeroUI (antes NextUI).

Diseño: Estética moderna basada en Glassmorphism.

Paleta de Colores: Principal #8906e6 (Violeta), Fondo #000 (Negro) y Acentos #FFF (Blanco).

Funcionalidades Actuales
Inventario: Gestión de productos con alertas de stock crítico.

Proveedores: Directorio de distribuidores.

Caja Rápida: Punto de venta optimizado para lector de barras.

Historial de Ventas: Registro de transacciones y rentabilidad.

Cartera de Clientes: Control de cuentas corrientes ("fiado") y deudas.

Objetivo de la Evolución (IA)
Implementar una capa de inteligencia (AI-as-a-Service) que:

Realice análisis predictivo de stock.

Ofrezca consejos financieros basados en el historial de ventas.

Actúe como un consultor de negocios para el usuario final.
